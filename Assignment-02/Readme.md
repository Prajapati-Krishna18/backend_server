# E-Commerce Product API

=> Build a REST API using Express.js that manages product data for an e-commerce platform using in an in-memory JSON array.

## Implemented Routes
    1. GET /products :- Return All products
    2. GET /products/:id :- Return product by ID
    3. GET /products/category/:categoryName :- Return products by category
    4. POST /products :- Add a new product
    5. PUT /products/:id :- Replace entire product    
    6. PUT /products/:id/stock :- Update only stock value
    7. PUT /products/:id/price :- Update only price

## Sample API URLs 
    http://localhost:5002/products
    http://localhost:5002/products/:id
    http://localhost:5002/products/category/:categoryName
    http://localhost:5002/products
    http://localhost:5002/products/:id
    http://localhost:5002/products/:id/stock
    http://localhost:5002/products/:id/price
